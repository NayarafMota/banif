# BANIF - Sistema Bancário

## Tecnologias
- AdonisJS
- React
- MySQL (XAMPP)

## Funcionalidades
- Login com autenticação
- Cadastro de clientes
- Criação de contas
- Transferência Pix
- Extrato bancário
- Aplicações financeiras

## Como rodar

### Backend
npm install
node ace serve --watch

### Frontend
npm install
npm start

### Banco de Dados
Importar o arquivo banif.sql no MySQL
